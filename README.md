# 🚀 Formação Desenvolvedor Full Stack Junior - 2026 (Grupo +praTI + Codifica)

Este repositório contém meus estudos, exercícios, tarefas e projetos desenvolvidos durante a formação **Desenvolvedor Full Stack Junior - 2026**, promovida pelo **Grupo +praTI** em parceria com a **Codifica**.

O objetivo do treinamento é formar profissionais capazes de desenvolver aplicações completas (front-end e back-end), utilizando boas práticas de programação, metodologias ágeis e ferramentas modernas do mercado.

---

## 🎯 Objetivo do Curso

Proporcionar formação gratuita e de qualidade para capacitar estudantes no desenvolvimento de software, com foco em empregabilidade e preparação para o mercado de tecnologia.

---



## 📚 Conteúdo Programático (Ementa)

A formação é dividida em **6 módulos**, contendo aulas e tarefas práticas.

---

### ✅ Módulo 1 - Introdução à Ciência da Computação (10 aulas)

- História da Computação e funcionamento dos computadores  
- Introdução à programação com JavaScript  
- Condicionais e laços de repetição  
- Arrays uni e bidimensionais  
- Funções e objetos  
- Estruturas de dados: listas, filas, pilhas e árvores  

📌 **Tarefas:** Exercícios de lógica e estruturas de dados com JS.

---

### ✅ Módulo 2 - Desenvolvimento Web: Frontend Essencial e Git (12 aulas)

- Git e GitHub (versionamento e colaboração)
- Fundamentos do desenvolvimento Web
- HTML semântico
- Formulários e tabelas
- CSS básico e avançado (Flexbox, Grid, responsividade)
- JavaScript aplicado ao Front-end
- Manipulação de DOM e consumo de APIs

📌 **Tarefas:** Construção de páginas HTML/CSS e projeto web completo.

---

### ✅ Módulo 3 - Desenvolvimento Web com React (14 aulas)

- Introdução ao React e configuração de ambiente
- Componentes, props e estado
- useEffect e ciclo de vida
- Context API (estado global)
- Formulários e eventos
- Consumo de APIs REST
- Rotas (React Router)
- CSS Modules e Tailwind
- Hooks personalizados
- Testes em componentes
- Deploy com Vercel
- Desenvolvimento de SPA completa

📌 **Tarefas:** exercícios + desenvolvimento de SPA final.

---

### ✅ Módulo 4 - Fundamentos de Java e Introdução ao Ecossistema Spring (20 aulas)

- Fundamentos da linguagem Java
- Estruturas de controle e repetição
- Arrays uni e bidimensionais
- Classes, métodos e encapsulamento
- Interfaces, herança e polimorfismo
- Estruturas de dados em Java
- Exceções e tratamento de erros
- Introdução ao Spring Framework e Spring Boot
- Controllers, Services e boas práticas
- Upload/Download de arquivos
- Threads e concorrência
- Testes unitários e integração

---

### ✅ Módulo 5 - Backend com Spring Framework e Bancos de Dados Relacionais (20 aulas)

- Conceitos de banco de dados relacional
- Modelagem e normalização
- SQL (DDL e DML)
- Consultas com JOIN, funções agregadas e subconsultas
- Índices e otimização
- API RESTful com Spring Boot
- Arquitetura MVC
- JPA/Hibernate
- Segurança com Spring Security
- Autenticação JWT e OAuth2
- Testes unitários e integração
- Documentação e deploy de API completa

---

### ✅ Módulo 6 - DevOps Básico e Introdução à Inteligência Artificial com Python (16 aulas)

- Conceitos de DevOps e ferramentas
- Docker e Docker Compose
- Deploy automatizado
- CI/CD com GitHub Actions
- Introdução à linguagem Python
- Estruturas de dados e controle de fluxo
- Funções e módulos
- Fundamentos de Inteligência Artificial
- Machine Learning
- Implementação do GPT-2
- Análise de dados com bibliotecas Python
- Modelos simples com Scikit-learn
- Consumo de APIs de IA em projetos Web

---

## 🧩 Projeto Final

O curso exige o desenvolvimento de uma **Aplicação Web Completa**, contendo:

### ✅ Requisitos Funcionais

- CRUD completo (Criar, Ler, Atualizar e Excluir)
- Autenticação e autorização de usuários
- Controle de versão com Git e GitHub

### ✅ Requisitos Técnicos (Não Funcionais)

- **Front-end:** ReactJS (responsivo e acessível)
- **Back-end:** Spring Boot (API RESTful)
- **Banco de Dados:** MySQL ou PostgreSQL
- Endpoints RESTful completos e seguros
- Testes unitários (**mínimo 70% de cobertura**)
  - Back-end: JUnit
  - Front-end: Jest
- Metodologias ágeis (Scrum/Kanban)

### 📦 Entrega do Projeto

- Código hospedado em repositório público no GitHub
- Estrutura organizada e commits consistentes
- Documentação técnica contendo:
  - Guia de execução local
  - Arquitetura da aplicação
  - Funcionalidades implementadas
  - Decisões técnicas e desafios enfrentados
- Demonstração ao vivo do projeto final
